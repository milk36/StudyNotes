# Idea 自带工具

## [Http Client](https://cloud.tencent.com/developer/article/1770689)

可以完美替代 `Postman`

示例脚本,[参考](https://bbs.huaweicloud.com/blogs/268840):

```js
### Get请求
GET http://127.0.0.1:10006/test_http?{{idstr}}={{id}}&{{namestr}}={{name}}
Accept: application/json

### Post请求

POST http://127.0.0.1:10006/test_http
Content-Type: application/json

{
  "id": 111,
  "name": "milk"
}

### Post请求

POST http://127.0.0.1:10006/test_http
Content-Type: application/x-www-form-urlencoded

{{idstr}}={{id}}&{{namestr}}={{name}}

```

* 环境变量

  `http-client.env.json` 配置文件，定义通用变量;

  `http-client.private.env.json` 配置文件，定义敏感变量。例如说，username/password 账号密码、token 访问令牌等等。

  ```json
  {
    "milk_test": {
      "namestr": "name",
      "name": "value",
      "idstr": "id",
      "id": "1234"
    }
  }
  ```
