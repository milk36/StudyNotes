### 修改镜像源

- ~~[yrm插件](https://learnku.com/articles/15976/yarn-accelerate-and-modify-mirror-source-in-china)~~
- 常规配置方法:
  - 查看当前使用的镜像源: `yarn config get registry`
  - 修改镜像源: `yarn config set registry https://registry.npm.taobao.org`
- npm 安装 `cnpm`

### nodemon

nodemon 是一个工具，可在检测到目录中的文件更改时自动重新启动节点应用程序，从而帮助开发基于 Node.js 的应用程序。

使用方式: `nodemon [your node app]`
